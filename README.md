
# 📱 Mobile Sales Dashboard

## Overview

This Power BI project analyzes mobile sales data to uncover trends in revenue, product performance, customer behavior, and sales distribution. Data transformation and cleaning were performed using Power Query before creating an interactive dashboard.

## Tools Used

* Power BI
* Power Query
* Microsoft Excel / CSV

## Dashboard Features

* Revenue Analysis
* Sales Trends
* Product Performance
* Customer Insights
* Interactive Filters and Slicers
* KPI Cards
